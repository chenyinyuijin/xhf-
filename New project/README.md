# 人工智能算法期中复习资料

本仓库用于整理人工智能算法课程的期中复习内容，包含课程讲义、TA Session 资料、习题整理、复习总结以及部分实验代码与 notebook，方便统一归档、查阅和在 GitHub 上展示。

## 资料内容

当前仓库主要包含以下几类内容：

- 课程讲义：`Lec1-IntroAI.pdf` 到 `Lec5-RL.pdf`
- 辅导课资料：`TA Session 1.pdf`、`TA Session 2.pdf`、`TA Session 3.pdf`、`TA_Session_4_MDP.pdf`、`TA_Session_5_RL.pdf`
- 复习资料：`midterm_review_lec1_to_lec5.pdf`、`ai_quiz_questions_answers.pdf`、`ml_quiz_questions_answers_set2_v2.pdf`
- 实验与练习代码：`RL_demo.ipynb`、`SVM_vs_MLP_Demo.ipynb`、`week_3_backpropogation.ipynb`
- 补充材料：`week3_tutorial.zip`、`wk5.zip`

## 适用范围

本仓库适合用于：

- 人工智能算法课程期中复习
- 平时作业与实验资料归档
- 课程学习过程展示
- GitHub 个人学习仓库整理

## 使用方式

### 在线查看

上传到 GitHub 后，可以直接在线查看 `README`，并下载 PDF、zip 和 notebook 文件。

### 本地使用

克隆仓库：

```bash
git clone <your-repo-url>
cd <your-repo-folder>
```

如果需要运行 notebook，建议使用：

```bash
jupyter notebook
```

或：

```bash
jupyter lab
```

## 仓库整理建议

如果你后续还会继续补充 PPT、作业和代码，建议逐步整理成下面这种结构：

```text
.
├── README.md
├── LICENSE
├── .gitignore
├── ppt/
├── homework/
├── notes/
├── notebooks/
└── code/
```

这样后续维护、展示和下载都会更清晰。
